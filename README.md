# Starten der Komponenten
### Frontend:
```
npm i
npm run serve
```

### Api:
In ./datenservice:
```
npm i
node index.js
```



Das Frontend und die Api werden zurzeit via Heroku bereitgestellt.
Die Datenhaltung geschieht via einer MongoDB-Datenbank.
